# Mid-to-Senior Engineer

**Reports to:** Tech Lead
**Function:** Engineering
**Headcount:** 2-3 in this role

## Core Responsibilities
- Ship features end-to-end: design discussion → code → tests → deploy → monitor
- Own a feature area or service for the duration of its lifecycle
- Participate in code review — give and receive
- Pair on hard problems with the tech lead and other engineers
- Write the docs for what you ship (AI-drafted, human-finalized)

## AI-Augmented Responsibilities
- **AI handles:** generating boilerplate, writing initial tests, refactoring patterns,
  drafting commit messages and PR descriptions, explaining unfamiliar code, suggesting
  edge cases, generating migration scripts
- **Human judges:** whether the structure is right, whether the abstraction earns its
  complexity cost, when an AI suggestion is right-shape-but-wrong-implementation, how
  to phase a change to minimize risk

## Key Skills to Hire For
- Has shipped features to real users (not just internal tools or side projects)
- Reads code as well as writes it — comfortable in unfamiliar codebases
- Strong debugging instinct — can isolate a problem to a single hypothesis quickly
- Treats tests as a thinking tool, not a chore
- Comfortable using AI for 30-70% of code production while owning the result

## Interview Signals (AI-native vs. traditional)
- **Strong:** Can walk through their AI usage pattern — when they reach for it, when
  they don't, what they review carefully
- **Strong:** Has shipped a recent project and can describe a bug they almost shipped
  but caught in review
- **Strong:** Has a strong opinion on PR size, test coverage targets, and rollout
  cadence
- **Weak:** Either prides themselves on never using AI, or has a workflow where AI
  output goes straight to PR with no edit
- **Weak:** Treats every problem as a new system rather than a small change

## Compensation Band (US, mid-range senior IC)
- Base: $175K - $220K
- Equity: 0.2% - 0.6% (early-stage)
- Total comp target: $240K - $320K
